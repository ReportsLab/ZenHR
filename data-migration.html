Upload the edited Excel file. The system previews changes first. Nothing is written to Firebase until you press Apply patches.

Edited Excel file Accepted: .xlsx, .xls. Rows with __importAction = skip will be ignored.

Rules used by this tool

- Export reads: employeeDirectory, submissions, loginIndex, editLogs. - Import never deletes documents. - Import never uses full replacement. - Import uses set(..., { merge: true }) / batch.set(..., { merge: true }). - editLogs is exported for backup only and is not imported back. - Blank cells only clear fields that already existed in the exported row. Blank cells for fields that did not exist originally are ignored. - __docId and __collection must remain unchanged for reliable patching.
